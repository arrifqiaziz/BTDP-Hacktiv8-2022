Exploration of Katalon Studio Main Features

SCRIPT MODE
Katalon menyediakan fitur script mode untuk melihat lebih jelas setap struktur test nya
setiap test flow yang di record manual langsung generate mode script nya, pun sebaliknya

Variable
Terdapat 2 jenis variabel dalam katalon
- Test Case Variable
- Global Variable

Test Case Variable
terdapat mode manual variable dan script mode variabel
setiap variabel dapat di inisiasi di keduanya
lalu masukan variable pada value di test flow nya
jalankan

Global Variable
berada pada Profile di Test Explorer
kita dapat menambahkan global variable profile sendiri
tambahkan variable seperti biasa
cara running nya adalah dengan memilih Select Execution Profile sesuai dengan tempat global variabel berada
