Sesi 08 - Integration with GIT and JIRA

Katalon sudah terinstall secara default build in GIT
sehingga tinggal konfiurasi akun GIT nya

Clone -> Untuk menyimpan project dari GIT ke local (jika file belum ada)
Share -> Untuk membagikan
Show History -> Untuk melihat riwayat vcs
Manage Branch -> untuk mengatur Branch
Commit -> Untuk menyiapkan file yang akan di push dengan comment
Push -> Untuk mengupload file dari local ke git
Pull -> untuk mengambil dari git ke local secara update
Fecth -> untuk update project 

Cara Push
klik Commit
drag and drop file yang akan di commit push
buat komentar perihal file
commit and push
