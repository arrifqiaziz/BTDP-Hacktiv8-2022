Introduction to Katalon Studio

https://katalon.com/download/#free

Katalon Studio adalah sebuah Automated Software Testing yang powerfull karena dapat melakukan testing pada Web App, API, Mobile App, dan Desktop

Dokumentasi Katalon
https://docs.katalon.com/

Katalon Workflow
- Initiate	-> terdapat banyak pilihan template untuk testing atau blak, juga dapat dengan mudah integrasi dengan Jenkins, GIT, dan JIRA
- Create	-> Membuat record, test case, automation test case
- Operate	-> Dapat mengoperasikan banyak test case sekaligus dengan cara running yang berbeda
- Report	-> Terdapat banyak format advance logging, debug data, automatic screenshot
- Maintain	-> Automatic update setiap ada perubahan object sehingga mengurangi potensi bug dan error


Instalasi Katalon
Setelah download, extract ke direktori yang diinginkan
Lalu buka katalon.exe

Test Scenario
Sebelum melakukan testing harus dibuat dokumentasi perihal Test Scenario atau apa saja test yang akan dilakukan.
hal-hal terpenting yang harus ada dalam test scenario adalah
- Test ID
- Functional Test
- Test Scenario
- Step
- Test Data
- Expected
- Actual
- Result
untuk hal lain semisal Positive/Negative Scenario atau Preconditional itu optional tergantung kebuthan

web target test https://katalon-demo-cura.herokuapp.com/

Setelah membuat test scenario, lanjut buka Katalon
new project
buat test case
record sesuai test scenario
jalankan
lihat status dan hasil test di log viewer
