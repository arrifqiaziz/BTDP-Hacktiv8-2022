Introduction to Java Languages

Bahasa Java adalah Bahasa pemograman high-level, dibuat oleh James Gosling yang
kemudian di akuisisi oleh Sun Microsystems tahun 1995. Banyak dijalankan pada variety
platforms Windows, MasOS , and UNIX

Bahasa Pemograman paling popular yang bersifat Independent Platform yang Simple, Secure,
Portable mendukung architecture-neutral dan juga yang paling penting adalah Object
Oriented Programming!

Cara Install Java
- Download file jdk
https://www.oracle.com/java/technologies/downloads/
- Lakukan instalasi seperti biasa (harus ingat dimana lokasi file)
- Setting Environtment Variable (evn) dengan cara
	- Copy direktori bin java
	- Buka env
	- Klik path -> new -> paste kan direktori bin java
- Cek status instalasi java dg cara
	- Buka cmd
	- ketik java -version
	- ketik javac -version

Syntax Java
Package	-> folder yang berisi sekumpulan program Java
Import	-> sekumpulan class dan fungsi yang bisa kita gunakan dalam membuat program
Class		-> class dideklarasikan sebagai nama program
Method Main	-> method utama yang akan dijalankan pertama kali
Input Output-> memasukan dan mengeluarkan data
