Operator Java
Di dalam java ada beberapa tipe Operator yang wajib dipahami , yaitu :
● Operator Aritmatika
● Operator Penugasan
● Operator Pembanding
● Operator Logika


*OPERATOR ARITMATIKA*
	1. Penjumlahan	+
	2. Pengurangan	-
	3. Perkalian	*
	4. Pembagian	/
	5. Sisa Bagi	%
	6. Increement	++
	7. Decrement	--

*OPERATOR PENUGASAN*
	1. Pengisian Nilai		=
	2. Pengisian dan Penambahan	+=
	3. Pengisian dan Penurangan	-=
	4. Pengisian dan Perkalian	*=
	5. Pengisian dan Pembagian	/=
	6. Pengisian dan Sisa Bagi	%=

*OPERATOR PEMBANDING*
	1. Lebih Besar		>
	2. Lebih Kecil		<
	3. Sama Dengan		==
	4. Tidak Sama Dengan	!=
	5. Lebih Besar Sama Dengan	>=
	6. Lebih Kecil Sama Dengan	<=

**OPERATOR LOGIKA*
	1. Dan (Keduanya harus true) maka bernilai true	&&
	2. Atau (Diantaranya harus true) maka bernilai true	||
	3. Negasi (kebalikan)		!
